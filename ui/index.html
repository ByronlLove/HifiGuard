Comment utiliser AutoEq ?
  1. Allez sur le site autoeq.app
  2. Cherchez le modèle exact de votre casque.
  3. Sélectionnez la cible Flat.
  4. Dans la section Equalizer, choisissez l'application EqualizerAPO GraphicEq.
  5. Téléchargez le fichier .txt généré.
  6. De retour ici, cliquez sur Associer et sélectionnez ce fichier.

Assurez-vous que le fichier est bien au format .txt pour GraphicEq.
